# YouTube Manager CLI

A command-line application built in Python to manage and organize YouTube-related data using a simple and interactive CLI interface.

---

## 🚀 Problem

Managing YouTube-related data manually (videos, notes, links, etc.) can become messy and unstructured.

There is no simple lightweight tool to:
- Store YouTube data locally  
- Access it quickly  
- Manage it via terminal  

---

## 💡 Solution

This project provides a **CLI-based tool** that allows users to:

- Store YouTube-related data  
- Retrieve and display saved entries  
- Manage structured data using a simple command-line interface  

---

## ⚙️ Features

- Add video entries  
- View all stored videos  
- Store data in a structured format (JSON file)  
- Lightweight and fast CLI interface  
- Persistent storage using local files  

---

## 🛠️ Tech Stack

- Python
- SQLite (database)  
- JSON (for data storage)  
- File Handling  
- CLI (Command Line Interface)  

---

## 🔄 How It Works

1. The application loads data from a JSON/text file  
2. User interacts through CLI options  
3. Data is:
   - Added  
   - Viewed  
   - Saved back to file  
4. Data persists between runs
